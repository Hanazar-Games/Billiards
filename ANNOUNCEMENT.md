# 3D Billiards v1.0.0 Announcement

3D Billiards v1.0.0 is the first full release of the browser-based pool game.

Highlights:
- Standard 8-ball play with local two-player and AI opponent modes.
- 9-ball mode, free play, challenges, achievements, stats, and shot replays.
- Three.js table rendering with cannon-es physics, spin controls, aim assist, shot trails, particles, and procedural audio.
- Clear local startup guidance for running the game through the Vite development server.

Run locally:

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.
